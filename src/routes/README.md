# PlaceMux Backend

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Authentication
- Swagger Documentation

## Run Project

npm install

npm run dev

## API Routes

POST /auth/send-otp

POST /auth/verify-otp

GET /profile

POST /profile

PUT /profile

DELETE /profile

## Swagger

http://localhost:5000/api-docs
